# Kernel Examples in This Folder

Each kernel folder in this directory usually contains around 10 example programs or more. These examples are provided mainly as references for development and extension.

They are **not all required** for the system to run. In normal execution, the program does not depend on having a large number of example files for each kernel.

The main purpose of keeping multiple examples is to show:

- how a kernel is organized,
- what kinds of inputs and outputs are expected,
- how the schema, description, dataset, and tests are prepared,
- and how a new kernel can be added following the same pattern.

In other words, these example files serve as demonstrations of the recommended way to implement and register new kernels, rather than as a strict runtime requirement.

If you want to add a new kernel, you can use the existing folders here as templates and follow the same structure.
